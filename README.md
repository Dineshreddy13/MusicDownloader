# 🎶 Music Downloader with Metadata & Lyrics

This is a Python tool to download songs using `yt-dlp`, from youtube music

---

## 📦 Features

- Download songs from YouTube music using `yt-dlp`
- Embed Spotify metadata into the song
- Fetch and embed synced lyrics
- Organize and save songs with proper formatting (m4a)
- Token-based Spotify API support
